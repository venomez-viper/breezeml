# breezeml-model

Model serving API generated by BreezeML. No breezeml dependency at runtime.

## Run locally

```bash
pip install -r requirements.txt
uvicorn app:app --host 0.0.0.0 --port 8000
```

## Run with Docker

```bash
docker build -t breezeml-model .
docker run -p 8000:8000 breezeml-model
```

## Predict

```bash
curl -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '[{"sepal length (cm)": 0.0, "sepal width (cm)": 0.0, "petal length (cm)": 0.0, "petal width (cm)": 0.0}]'
```

Interactive docs: http://localhost:8000/docs
